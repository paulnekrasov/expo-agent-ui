# Ecosystem Research Proposals

This directory contains structured proposal documents generated by the Ecosystem Research
Autonomous Agent Loop (`docs/agents/ECOSYSTEM_RESEARCH_AUTONOMOUS_AGENT_LOOP_PROMPT.md`).

Each file follows the naming convention: `YYYY-MM-DD-ecosystem-research.md`

Proposals are review-only artifacts. No source code, dependency, or configuration changes
should be made based on a proposal until the human reviewer has approved the specific items.

## How To Use

1. Read the Executive Summary first.
2. Review Critical Findings before anything else.
3. Approve, reject, or defer each finding individually.
4. For approved findings, create a bounded task in `docs/agents/TASK.md`.
5. For approved doc-only updates, the research agent may have already self-implemented them.
